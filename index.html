What is Code Drone?

Code Drone is an AI-assisted coding utility that helps programmers speed up their coding through contextual suggestion, pattern recognition, and real-time testing, to name a few.

How does it work?

Code drone works as a software layer on your system, it automatically interfaces to any browser, text editor, IDE and, command-line tools on your system. It operates on its own set of independent processes, utilizing the existing input/outputs and multimedia on the system. It has various configurations available so the user can tweak the level of feedback the AI offers, it can also work in conversation mode if you just want to talk through a problem, or you can switch to coaching mode, where the AI gets more involved with your code and might offer clues or hints

What are people saying about code drone?

So far the feedback has been overwhelmingly positive. For example one of our fields test reported that out of 120 programmers who spent 3 months with code drone, 91% of them grew in their skills faster than their colleagues who did not use code drone. That’s pretty amazing. Employers who paired code drone with programmers reported higher productivity metrics across all development areas. 88% of programmers who worked with code drone for 30 days reported that they enjoyed the experience and actually preferred it.
